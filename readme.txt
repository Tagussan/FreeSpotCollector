FREESPOTの全スポットの緯度と経度を収集します。
使い方
CollectFreeSpots.rb -> 実行すると、FreeSpotsフォルダに各都道府県のFreeSpotのデータがxml形式でダウンロードされます。
　緯度、経度、住所、スポット名、いろいろはいってるのでそのまま好きにお使い下さい。
CreateOruxMapsKML.rb -> 実行すると、FreeSpotsForOruxMapsKMLフォルダにOruxMapsで使用出来るkmlオーバーレイが作成されます。そのままOruxMapsの所定のフォルダに入れてお使いいただけます。

Ubuntu12.10で動作確認済み。
